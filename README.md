# rn-local-p2p

Peer to peer connection library over a local network, WiFi, without internet access using UDP

## Installation

```sh
npm install rn-local-p2p
```

## Usage


```js
import { multiply } from 'rn-local-p2p';

// ...

const result = multiply(3, 7);
```
